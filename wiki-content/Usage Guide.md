# Usage Guide

## Opening the calculator

The application is a standalone HTML page.

1. Download or clone the repository.
2. Open `index.html` in a modern web browser.
3. Wait for the page to load fully.

## Inputs you can modify

### Efficiency
- Automation Integrity efficiency gain

### Labor and Operations
- Engineer labor rate
- Work days per year
- Items researched per day
- Number of DCS sites

### Baseline Effort
- Research / determine tag attributes
- Control strategy diagrams
- Engineering notes research
- Dependency research
- Control programs / spares / issues
- Preparation for MoC or export

### Investment
- Software cost (Year 1)
- Support cost (Year 1)
- Support escalation percentage
- Service cost (Year 1)
- Analysis horizon in years

## Outputs

The dashboard updates automatically and includes:

- Annual savings per DCS
- Net savings over the analysis horizon
- ROI percentage
- Payback period
- Effort comparison table
- Annual savings table
- Investment schedule
- ROI summary
- Multi-site totals

## Exporting a report

Use the **Download Word Report** button to generate a `.docx` report based on the current scenario.
