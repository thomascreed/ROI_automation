# Automation Integrity ROI Calculator

Welcome to the project wiki for the Automation Integrity ROI Calculator.

## What this project does

This project provides a browser-based ROI calculator for estimating the business impact of Automation Integrity adoption. It is designed as a standalone HTML tool with no build system required.

## Main capabilities

- Compare baseline engineering effort vs. automated effort
- Estimate annual labor savings
- Model multi-year investments
- Calculate ROI and payback period
- Generate a Word report for the active scenario
- Roll up savings across multiple DCS sites

## Project structure

- `index.html` — calculator UI, styling, and JavaScript logic
- `README.md` — repository overview and local usage instructions

## Quick start

1. Open `index.html` in a browser.
2. Adjust assumptions.
3. Review KPI outputs and financial tables.
4. Export the report if needed.
